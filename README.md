# 🚀 SculptCV – Resume Builder

SculptCV is a **full-stack Resume Builder SaaS application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. It allows users to create, manage, and export professional resumes with a clean and responsive interface.

---

## 📌 Project Overview

This project demonstrates real-world SaaS development including authentication, dynamic dashboards, resume editing, and PDF generation. It is designed as a portfolio-ready full-stack application showcasing modern web development practices.

---

## ✨ Key Features

* 🔐 Secure User Authentication (JWT-based login & signup)
* 📊 Dynamic Dashboard for resume management
* 📝 Interactive Resume Builder (multiple sections support)
* 📈 Resume completion progress tracking
* 🖼️ Profile image upload functionality
* 📄 Export resumes as downloadable PDF
* 📱 Fully responsive design for all devices

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Durgesh-https/SculptCV.git
```

### 2. Install dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3. Configure environment variables

Create a `.env` file inside `/backend`:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=your_port
```

---

### 4. Run the application

```bash
cd backend
npm start
```

Then start frontend:

```bash
cd frontend
npm run dev
```
